# SimpleCalc

Ein einfacher Taschenrechner für die Konsole, geschrieben in C#. Mein zweites richtiges C# Projekt.

## Was er kann

Grundrechenarten: +, -, *, /. Division durch 0 wird abgefangen, ungültige Eingaben auch.

## Beispiel

```
╔══════════════════════════╗
║      SimpleCalc v1.0     ║
║     by lowkyloki         ║
╚══════════════════════════╝

Operationen: + | - | * | / | exit
Wähle eine Operation: *
Erste Zahl: 12
Zweite Zahl: 5

Ergebnis: 12 * 5 = 60
```

## Starten

```bash
dotnet run
```

## Technologien

C#, .NET, Konsolenanwendung
